## Setting up PyCharm on Linux:
- Download Pycharm
- Add to ~/.bashrc
  - `export PATH=$PATH:/path_to_pycharm/PyCharm/bin/`
- run `source ~/.bashrc`
- run Pycharm in Project directory using terminal `./pycharm.sh`
